# UNO Multiplayer v2.0

Full-featured online multiplayer UNO game with animations, themes, AI bots, and sound effects.

## Deploy to Render.com

1. Push all files to GitHub
2. Connect repo to Render.com
3. Build: `npm install`, Start: `npm start`
4. Free tier works perfectly

## Features
- Real-time multiplayer with room codes
- AI bots for practice
- Score tracking across rounds
- 5 card themes (Classic, Neon, Dark, Ocean, Nature)
- Spectator mode
- Sound effects for every action
- Card hover/select/play sounds
- Special card sounds (Skip, Reverse, +2, +4, Wild)
- Rich animations on all pages
- Floating particles in lobby/room
- Card fly animations when played
- Emoji reactions
- In-game chat
- Turn timer with auto-draw
- Fully responsive (desktop + mobile)

## Keyboard Shortcuts
- U = Say UNO
- D = Draw card
- Esc = Close overlays
